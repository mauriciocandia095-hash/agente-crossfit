# Agente de Entrenamiento CrossFit

Aplicación simple creada con Python, Gradio y Gemini que genera un plan de entrenamiento de 5 días basado en el nivel del usuario y posibles molestias o lesiones.

## Tecnologías
- Python
- Gradio
- Gemini API
